# Éditeurs et canevas

Les éditeurs et canevas sont des surfaces complémentaires, pas une fonctionnalité vedette unique.

| Surface                 | Propriété actuelle                                                             |
| ----------------------- | ------------------------------------------------------------------------------ |
| Éditeur riche           | Toolkit, avec réexportations de compatibilité depuis Core.                     |
| Blocs                   | Core durant cette phase.                                                       |
| Design                  | Patron de modèle avec `design-tweaks`, pas un éditeur prêt à l'emploi complet. |
| Présentation de canevas | Composition Toolkit ou application.                                            |

Importez l'éditeur depuis `@agent-native/toolkit/editor`. Les anciens chemins Core restent disponibles comme réexportations de compatibilité. Gardez schémas, commandes et persistance dans l'app. Les blocs restent dans Core, mais sont documentés ici car la section Toolkit est le catalogue de l'interface réutilisable.
