# Doctor

`agent-native doctor` exécute huit contrôles: les sept contrôles de sécurité et `migration-manifest`. Celui-ci détecte les imports présents dans le `migration-manifest.json` installé et n'a pas d'opt-out. Lancez `agent-native doctor --only migration-manifest`, prévisualisez avec `npx @agent-native/core@latest upgrade --codemods`, puis appliquez avec `--yes`. La sortie JSON liste les huit noms dans `guardsRun`.

Les entrées marquées `planned` sont uniquement signalées comme avertissements et ne sont pas réécrites. Les entrées actives sont des constats; le codemod ne les réécrit que si l'export de destination est installé.
