# エディターとキャンバス

エディターとキャンバスは関連する表面であり、単一の主力機能ではありません。

| 表面             | 現在の所有                                                                               |
| ---------------- | ---------------------------------------------------------------------------------------- |
| リッチエディター | Toolkit。Core から互換性のために再エクスポート。                                         |
| ブロック         | この段階では Core。                                                                      |
| Design           | `design-tweaks` を持つテンプレートパターン。完全なドロップインエディターではありません。 |
| キャンバス表示   | Toolkit またはアプリの合成。                                                             |

エディターは `@agent-native/toolkit/editor` からインポートします。従来の Core パスは互換性のための再エクスポートとして残ります。スキーマ、コマンド、永続化はアプリに置きます。ブロックは Core に残りますが、Toolkit セクションは再利用可能 UI のカタログなので、ドキュメントはここにまとめます。
