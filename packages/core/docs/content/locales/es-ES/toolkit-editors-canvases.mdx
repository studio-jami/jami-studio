# Editores y lienzos

Los editores y lienzos son superficies relacionadas, no una única función principal.

| Superficie             | Propiedad actual                                                           |
| ---------------------- | -------------------------------------------------------------------------- |
| Editor enriquecido     | Toolkit, con reexportaciones de compatibilidad desde Core.                 |
| Bloques                | Core en esta fase.                                                         |
| Design                 | Patrón de plantilla con `design-tweaks`, no un editor instalable completo. |
| Presentación de lienzo | Composición de Toolkit o de la app.                                        |

Importa el editor desde `@agent-native/toolkit/editor`. Las rutas antiguas de Core siguen funcionando como reexportaciones de compatibilidad. Mantén en la app los esquemas, comandos y persistencia. Los bloques siguen en Core, pero se documentan aquí porque la sección Toolkit es el catálogo de UI reutilizable.
